# intelligent-automation-portfolio
A curated portfolio of Intelligent Automation, AI, Machine Learning, and Data Analytics projects focused on transforming traditional process automation into AI-driven, cost-optimized systems.

## 🤖 AI-Driven Automation
- **[AI Support Call Intelligence Engine](https://github.com/madakkuni/ai-support-call-intelligence-engine/tree/main)** is an **AI-powered platform** that transforms support call recordings into performance insights, quality metrics, and actionable recommendations to **improve service desk operations** and **customer experience**.

The platform processes enterprise support call recordings using audio cleaning, speech-to-text transcription, AI-driven conversation analysis, and automated call quality scoring.
